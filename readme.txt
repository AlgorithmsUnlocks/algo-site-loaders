This plugin allows you to set a custom loader for your website's frontend.

== Description ==

Algo Site Loaders is a simple loader plugin that can create awesome loader animations for your sites.

== Frequently Asked Questions ==

= How can I change the loader animation? =

You can change the loader animation by accessing the plugin settings in the WordPress dashboard. There, you'll find options to select a new animation and adjust various settings.

= Can I customize the loader's appearance? =

Yes, the plugin provides customization options allowing you to tweak the loader's appearance, such loader type at this moment, but the color, size, and style are not available now.

== A brief Markdown Example ==

**Features:**

1. Set custom loader animation for your website.
2. Easy-to-use settings panel.
3. Customization options for the loader's appearance.

**Installation:**

* Download the plugin ZIP file.
* Go to your WordPress admin dashboard.
* Navigate to Plugins > Add New.
* Click on "Upload Plugin" and select the ZIP file.
* Activate the plugin.

**Usage:**

1. Once activated, go to the plugin settings in the WordPress dashboard.
2. Choose or upload your desired loader animation.
3. Customize the loader's appearance and behavior settings.
4. Save your changes.
